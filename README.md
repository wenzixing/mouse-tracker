# Mouse Kinematics Analysis Tool (认知康复评估: 鼠标运动学分析工具)

A Python-based tool for assessing mouse control capabilities using **Fitts' Law** principles. Designed for cognitive rehabilitation assessment and motor control research.

## Features
- **Fitts' Law Testing**: Simple "Start -> Target" clicking tasks.
- **Kinematics Analysis**:
  - Trajectory Curvature (Path Efficiency).
  - Reaction Time & Total Movement Time.
  - Movement Speed (px/s) & Pauses.
- **Visualization**: Trajectory plotting with comparison to ideal straight-line paths.

## Installation

1.  Clone the repository:
    ```bash
    git clone https://github.com/YOUR_USERNAME/mouse-tracker.git
    cd mouse-tracker
    ```

2.  Install dependencies:
    ```bash
    pip install -r requirements.txt
    ```

## Usage

Run the main script:

```bash
python mouse_tracker.py
```
